项目目录
------
本项目为bobetax-cli 项目的开发文档。

### 项目目录说明
```
├── readme.md
├── bin
│   └── install.js
├── package-lock.json
├── package.json

├── src
│   ├── component.js
│   ├── create.js
│   ├── download.js
│   ├── echo.js
│   ├── main.js
│   ├── test.js
│   └── utils.js
├── tree.js
└── yalc.lock
```
### 启动
  * use yarn
```
  yarn add @bobetax/cli -g
```
  * use npm
```
  npm install @bobetax/cli -g
```
  * use pnpm
```
  pnpm install @boetax/cli -g  
```


### 使用
```
bobetax-cli create "project-name"
```
